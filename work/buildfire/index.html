BuildFire Plugin Work

Custom BuildFire plugins we've built and documented.

Our history with BuildFire goes back further than plugin work: our team worked as leading engineers on early BuildFire platform development, before founding PixelPlugins. Since founding PixelPlugins, that's continued as custom plugin development for BuildFire apps — including two builds documented as public setup guides below, real configuration steps for plugins we designed and shipped, not marketing copy about them.

AI Chatbot Plugin

An AI chatbot plugin for BuildFire apps, backed by an OpenAI vector store. See the full API key setup guide for the configuration steps.

HIPAA Dynamic Form Builder

A dynamic form builder plugin built for HIPAA-relevant data collection inside a BuildFire app — configurable fields, previewing, saving, and email integration. See the full setup guide.
